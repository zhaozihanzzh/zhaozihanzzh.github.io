Python3 整除的浅思

Python 3 中，采用 // 来实现整除，两个整型变量整除后得到的变量依然是整型。而采用普通的除号 / 得到的却是浮点型。

Python 3 中，整型支持可以做到硬件允许范围内的大，但浮点型没有这样的实现机制。为了保留相对完整的大整数运算能力支持，需要整除的存在来避开对浮点的依赖。

那么，为什么两个整型采用除号运算，得到的却是浮点型？
